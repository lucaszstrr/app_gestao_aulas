# App Gestão Aulas Particulares

A web application for managing private lessons for a local education company. This system helps administrators, teachers, and responsibles of students organize schedules, and payments.

## Features

- 📅 Lesson scheduling and calendar management
- 👨‍🏫 Teacher and student profiles
- 💰 Payment tracking and invoicing
- 📊 Performance analytics and reporting
- 🔐 User authentication and role-based access

## Technologies Used

- **Backend**: Laravel, PHP
- **Frontend**: CSS, JavaScript
- **Database**: mySQL

## Installation

1. Clone the repository:

   cd app_gestao_aulas
