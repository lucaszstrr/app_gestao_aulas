# 📚 Sistema de Gestão de Aulas Particulares

Este projeto foi desenvolvido para uma empresa de aulas particulares com o objetivo de facilitar o **registro de aulas**, o **acompanhamento da presença de alunos** e o **controle dos pagamentos semanais realizados aos professores**.

O sistema é utilizado no dia a dia pelos professores, promovendo mais **organização, agilidade e transparência** no acompanhamento das aulas ministradas.

---

## ✅ Funcionalidades Principais

- Cadastro e login de professores pelo admin
- CRUD completo de alunos
- Sistema de geração de código PIX para pagamento de aluguel das salas
- Sistema de mensagem de cobrança para pais e responsáveis
- Dashboard de admin para visualizar a quantidade de aulas
- Geração de planilhas para maior comodidade
- Painel individual para cada professor

---

## 🛠️ Tecnologias Utilizadas

- **PHP 8.1**
- **Laravel 10**
- **MySQL**
- **Blade** (sistema de templates do Laravel)
- **Eloquent ORM**
- **Validações nativas do Laravel**
- **Controle de acesso com middleware**
